# wm-argumentative 팩 (슬림 v1.0)

> 공개판 정화 고지 — 원본의 "스니펫(verbatim)" 절은 외부 저작물(신문 칼럼 원문) 인용이므로 공개판에서는 각 스니펫의 **구조·기능 설명판**으로 교체했다. 원문 문구는 담지 않는다(구조·리듬만 기술). 레지스터·리듬 수치·구조 관습·대조페어·Do-NOT 등 저자 자체 분석은 원본 그대로 보존.

> 장르 규범(신문 칼럼·논설 일반). 근거: reference/argumentative 대표 10편(보수·진보·중도·철학·정책·과학 고른 표집). 정체성 한 줄 — 논지를 선명히 세우고, 반론을 미리 끌어안고, 결어에서 강도를 최고로 올린다.

## 레지스터
- 기본 종결 평서형 `-다`(단정·판단 전부 눌러 담음). "~것이다" 남발 금지, 강조로만.
- 추정·유보는 정밀 조절: `-ㄹ 것이다 / -ㄹ 만하다 / -았을 법하다 / -았으리라 / -인지도 모른다`. 확신↔유보 낙차가 논증 도구.
- 경어체(`-습니다`) 금지 — 대중 향한 평어체 공적 발화.
- 자기지칭: 해당 없음(화자는 대개 숨음, "나는" 앞세우지 않고 판단을 문장으로 제시).
- 호명: 독자를 "우리"로 묶어 공동 책임의 자리로 끌어들임.
- 낙차 규칙: 긴 문장 뒤 짧은 개념어 봉인구 1줄로 리듬 낙차. 결어에서만 온도 최대.

## 시그니처
| 표현 | 기능 | 빈도 |
|---|---|---|
| `A가 아니라 B다` | 대립쌍 논점 재정의 | 상 |
| `물론 X. 그러나 Y` | 양보-반전 반론 선점 | 상 |
| 유비 병치(과거·타국·타분야) | 판단을 독자에 넘김 | 상 |
| 단문 봉인구("이렇게 무섭다") | 문단을 개념어로 닫음 | 상 |
| 수사의문 결어("~생각하는가") | 답 정해진 물음으로 몰아세움 | 중 |
| 열거 점층("~한 자, ~한 자") | 문제 규모 체감 | 중 |
| 연쇄 자문("무엇인가?") | 논지 다음 단계로 밈 | 중 |
| 인용-판정 결합 | 권위 인용 직후 필자 판정 | 중 |
| 기원·명령 클로징("건투를 빈다") | 결어에서만 화자 전면 | 중 |
| 개념어 반복 타격(형해화·기득권·중독·먹이사슬) | 논증 뼈대 | 상 |

## 리듬
- 분량 1,300~1,800자, 문단 6~10개. 문장 길이 변주: 논거는 길게 쌓고 봉인·결어는 단문으로 끊음.
- 줄바꿈: 6막 배치마다 문단 분리, 반론·재확인·결어는 독립 문단.
- 구두점: 근거에 수치·고유명사·날짜 촘촘히("700% 폭등" "월급 144만6000원" "2만7317건"). 비유는 결정적 대목에만.

## 구조 관습
- 도입(훅): (a)시사 앵커 (b)일반명제 선포 (c)개인 일화·도발 중 하나. 첫 문장 추상어 금지.
- 전개: 논지를 2~3문단 안 대립쌍으로 못 박음 → 사례·통계·판결·외신·유비 층층이 축적 → "물론~그러나~"로 반론 선점 → 개념어로 재봉인.
- 마무리: 온도 최대. 수사의문 / 기원·명령 / 도입 회수 중 하나, 짧은 단문으로 끊어 여운.

## 대조페어 (금지 → 허용)
- `~에 있어서` → 조사 직결("한일관계에서는")
- `~로부터` (수동 출처) → 능동 주어로 재구성
- `~에 의해/의하여` → 능동태 주어
- `~을 가지다`(have 직역) → 서술동사로("영향을 준다")
- `~들` 복수 남발 → 무표지 또는 열거
- `~되어진다 / 보여진다`(이중피동) → `-다` 단정 능동
- `~에 다름 아니다` → "~일 뿐이다 / 바로 ~다"
- `아무리 ~해도 지나치지 않다` → 단정 강조문
- `~라고 할 수 있다 / ~인 것 같다 / ~라고 생각된다` → `-다` 단정
- 양비론·"균형 잡힌 시각" → 편을 드는 대립쌍 논지

## Do-NOT
- 판단·주장 문장에 피동 쓰지 마라 — 피동은 사건 자체가 수동일 때만("치여 숨졌다").
- 반론 문단 생략 금지 — "물론~그러나~" 빠지면 논증이 아니라 선동이 된다.
- 유비를 "이건 마치 ~와 같다"로 풀지 마라 — 두 사례를 병치하고 판단은 독자에 넘겨라.
- 첫 문장에 추상어·밋밋한 개괄 도입 금지.
- 접속사("그리고/또한/따라서") 남발 금지.

## 패턴뱅크 (구조 기술 — 문구는 새 소재로 창작)
- **S1 유비로 현재 심판** (C-argumentative-001): "잘못된 제도는 중독처럼 작동한다"류 일반명제를 먼저 세우고, 그 명제를 입증하는 실패 국가·역사 사례를 연속 2개 병치한 뒤 현재 대상으로 회귀하는 도입. 판단은 병치가 대신하고, 마지막은 개념어(중독·말로) 한 단어로 봉인.
- **S2 양보-반전 반론 선점** (C-argumentative-004): "물론 (최악의 극단)은 아니라고 믿는다"로 상대 입장을 먼저 넉넉히 인정 → "그러나 (본질·체감)에서는 다르지 않다고 본다"로 반전. 반박당할 극단 사례를 스스로 완충으로 깔아 현재 사안을 그 강도에 겹쳐 놓는 2단 구조.
- **S3 대립쌍 정식화 + 개념어 판정** (C-argumentative-006): 논란의 본질을 "A(표면·기술적 프레임)가 아니라 B(상위·철학적 프레임)이다" 한 문장으로 재정의. 상대가 세운 프레임을 부정하고 더 높은 층위로 논점을 격상시키는 판정문.
- **S4 단문 봉인 + 수사의문 결어** (C-argumentative-016): 사회적 약자가 진 고통의 규모를 짧은 진단 단문으로 못박고, 개념어(먹이사슬)를 반복 타격한 뒤, 답이 이미 정해진 수사의문("~라고 생각하는가")으로 독자를 몰아세워 결어를 닫는다.
- **S5 연쇄 자문 + "우리"로 묶는 결어** (C-argumentative-012): "다음 단계는 무엇인가"류 자문을 던져 논지를 다음 국면으로 밀고, 결어에서 주어를 "우리"로 전환해 공동 책임의 자리로 독자를 끌어들이며 출구 미결의 여운으로 마감.
