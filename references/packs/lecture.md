## 대조페어 (금지 → 허용)
1. **음슴체(`~함/~음/~됨`) 생성 금지 유지** — 단, 명사종결 자체는 0이 아니다(bare 0.033): 텍스트 소제목·용어 정의 명사문(`자기장의 변화로 전류와 전압이 유도되는 현상. 패러데이 법칙으로 표현`[B-107])·출처 라인으로만 허용.
2. 마크다운 `#`소제목·볼드(0.000) → 플레인 텍스트 한 줄 소제목 + 번호 리스트. 구 팩 "소제목 분절 금지"는 폐기 — 분절은 하되 장식 없이.
3. 전 문장 합쇼 통일 → 합쇼(정의)·죠/까요(동행)·반말 혼잣말 3중 직조(해요 0.358·해체 0.399).
4. 문장마다 마침표 반듯이 → 준종결로 흘려 잇기 + 초단문 낙차(편내 std 5.97).
5. 질문 하나 후 즉시 정리 → 질문 연달아 쏟고 답 유예(`모터가 뭔가요? 우선 GPT에게 물어봅시다`[B-107], 의문종결 0.148).
6. 어려운 대목 완곡 생략 → `~정도만 이해하시면 됩니다`로 난이도 선 긋고 진행(37회).
7. 추상 "AI" 지칭 → 실제 모델명 대구 + 의인화(`그록과 퍼플렉시티가 보조할 거예요`[B-224], `녀석` 89회).
8. 슬로건 결론·`정리하자면`(2회) 재요약 → 다음화 예고·링크·`-저자 배상-`으로 뚝 끊기. ※ `결론적으로`는 본문 중간 인과 수렴으로 6/64편 실사용 — 마무리 공식으로만 금지.
