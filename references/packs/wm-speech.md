# wm-speech 팩 (슬림 v1.0)

> 장르 규범 프로필 — "잘 쓴 한국어 연설문"의 공유 관습. 축: 청중 호명 · 리듬 반복(anaphora·삼항 병렬) · 클라이맥스(진단→전환→호소). 표본: C-speech-002/003/004/006/009/010/011/012/015.

## 레지스터
- 화계 분포: `-습니다/-ㅂ니다` 절대우세(문장 90%+). 변별 종결 3종 — 의지/약속 `-겠습니다`(후반 연쇄), 청유/결속 `-ㅂ시다`, 최후호소 `-십시오/-바랍니다/-부탁드립니다`. 조사(弔詞)만 예외로 `-십니까/-가십시오`(고인 직접 호명).
- 자기지칭: `저`(겸양) — 제가/저는/저의. 권위체 "본인/우리 정부"는 최소화.
- 호명: 청중을 **직접 부른다** — "여러분/국민 여러분/신입생 여러분/동료 의원 여러분", 격식 시 위계순 나열. 같은 호명구를 문서 내 3~5회 되풀이해 단락의 관절로.
- 낙차 규칙: `저 → 우리`로 미끄러지는 순간이 설득 지점. 클라이맥스로 갈수록 종결어미 상승(`-겠습니다`→`-ㅂ시다`→`-십시오`), 문장 길이는 하강 — **짧아지면 고조된다.**

## 리듬
- 단문 우세. 주장은 짧게 끊고, 배경·서사 구간만 만연체 장문 허용 — 핵심·클라이맥스로 갈수록 문장이 짧아진다(길이 하강=고조).
- 명사 종결(체언 마침)로 못을 박는다 — "그냥 ~입니다. ~일 뿐입니다." 꼴 반복 격하(실물은 스니펫 S1).
- 반복은 3의 배수(삼세번)가 기본값. 은유 씨앗을 도입에 심고 클라이맥스에서 회수.
- 채널: 연설문은 낭독체 — 문단 단위 호흡, 개조식 불릿 금지. 수사의문 `?`, 대구 쉼표로 낙차 표시.

## 구조 관습
7-비트 아크(짧은 연설은 압축):
- **도입** — ①호명 인사(위계순) ②겸양·자기 정위(화자 낮춰 눈높이).
- **전개** — ③상황 진단(숫자·역사·일화로 위기감) ④전환점("그러나/그럼에도"에서 핵심 메시지 첫 명시) ⑤논증·병렬("첫째/둘째"), 각 블록은 호명으로 재개.
- **마무리** — ⑥클라이맥스(문장 단축·종결어미 상승·수사의문·은유 수렴) ⑦코다(축원 한 줄+"감사합니다."+날짜·직함·이름 서명).

## 대조페어
- `~에 다름 아닙니다` → `~와 다르지 않습니다 / 사실상 ~입니다`
- `~에 있어서 / ~에 대하여`(남용) → 조사 `-에서/-에/-은`
- 이중피동 `-되어지다 / -지게 되다` → 능동
- `관심을 가지다`(have 번역투) → `관심을 두다/기울이다`
- `~로부터`(남용) → `~에게서/~에서`
- 영어 관용 직역("하루의 끝에") → 우리말 관용
- 균질한 중문 나열 → 문장 길이 파동(배경 장문 / 핵심 단문)
- 개조식 불릿·번호 목록 → 산문 병렬("첫째…둘째…") 또는 삼항 한 문장
- 진공 추상어("불평등이 심각합니다") → 구체 앵커(숫자·고유명사·장면 하나)
- `-이다/-한다/-일 것이다` 문어체 종결 → 격식 존댓말 `-습니다` 계열
- 청중 부재의 독백 → 호명구 삽입 + 주어 `우리`로 결속

## Do-NOT
- 개조식·불릿 나열로 논지를 진열하지 말 것(낭독 불가) — 반드시 산문 병렬로 녹인다.
- 호명 없이 시작·전개하지 말 것. 도입과 각 주제 블록 앞에 호명구 필수.
- 추상 가치어(정의·평화·통합)를 구체 명사·숫자·고유명사 없이 홀로 띄우지 말 것.
- 클라이맥스를 길고 균질한 문장으로 끝내지 말 것 — 단문+명사종결로 봉인.
- 번역투(§대조페어) 잔류 금지 — 서명 직전 전수 스캔.

## 패턴뱅크 (구조 기술 — 문구는 새 소재로 창작)
- **S1 격하→명명 3단** (C-speech-006): 같은 인물을 "그냥 ~입니다" 꼴로 두 번 격하한 뒤, 세 번째 문장에서 구체 수치를 붙여 은유 개념으로 승격·명명. 격하가 깊을수록 명명이 튄다.
- **S2 2인칭 정면 단정** (C-speech-011): 청자를 "당신"으로 지목해 통념의 지위를 부정하고, 반대 지위를 명사종결 2연으로 못박는 선언.
- **S3 미래 선언 삼단** (C-speech-009): 주어만 바꾼 동형 문장 3연 "~할 것입니다" — 약속 각인. 삼세번 리듬의 대표형.
- **S4 문두 반복 애도** (C-speech-012): 고인 직접 호명 + "얼마나 ~셨습니까" 수사의문을 문두 반복 — 조사(弔詞) 전용.
- **S5 청유 클라이맥스** (C-speech-003): "-ㅂ시다" 동형 청유 2연으로 결속 마감.
