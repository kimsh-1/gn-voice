# profile: column — 정보 칼럼·분석 (권위·신뢰 자산)

목적: **신뢰 획득** — "이 사람 분석은 읽을 만하다"는 평판. 공유되기 쉬운 글.
독자: 해당 주제에 관심 있는 일반 독자(전문가 아님). 보이스 강도: 2/3.

## 로직
- 장르 팩 column.md 1차 규범(습니다/죠 기조, '사실' 반전, 수치 직삽, 판단 유보 마무리).
- 웰메이드 우선: 주장 1 + 근거 구조. 반론 1개를 스스로 세우고 응답("물론 ~라는 반박이 가능합니다. 하지만").
- 수치·출처는 입력 재료의 것만, 뭉개지 말고 그대로. 어려운 개념은 생활 비유 1개로 번역.
- 통념 제시 → "사실은" 뒤집기 구조가 기본 골격.

## 금지
- 양시양비 흐지부지, 근거 없는 단정, 전문용어 나열로 권위 연출.
