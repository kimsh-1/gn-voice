# profile: launch — 출시 공지·모집문 (신규 강의·전자책·이벤트 알림)

목적: **알림 + 신청 전환**. copy보다 정보 비중이 높다 — 팬덤은 이미 신뢰가 있으니 팔기보다 정확히 알린다.
독자: 기존 팔로워·구독자(warm). 보이스 강도: 2/3.

## 구조 공식
1. **근황 훅**: 공지를 사담처럼 연다("한동안 조용했던 이유가 이겁니다").
2. **무엇을 만드는가**: 결과물의 실체 — 커리큘럼·목차·분량을 구체로.
3. **누구에게 필요한가 / 필요 없는가**: 필요 없는 사람을 먼저 걸러주는 정직함이 공냥 신뢰 자산.
4. **핵심 정보 블록**: 일시·가격·정원·신청 방법 — 평문 번호로 명확하게(이 프로파일에서는 정보 목록 허용).
5. **CTA 1개** (voice-core §4 CTA 금지 해제 — 행동 지시형만).

## 규칙
- 실적·경력 자백은 증거로 1~2개만(카피처럼 몰아넣지 않는다).
- 화계: 습니다체 기조 + 사담 구간 반말 낙차 1회.
- 날짜·가격·정원은 입력 재료의 값 그대로(날조·누락 금지 — 이 프로파일의 명제 보존은 엄격).

## 금지
- 과장 수식, 긴박 연출("서두르세요"), 정보 없이 감성만 채운 공지.
